# B2 — NO PURPOSELESS COLLECTION OR EXECUTION

Every job — collection, build, or analysis — has a defined purpose and
output category assigned before it begins. The system will not execute
an undefined job.

**This is the boundary that A1 (Concept Isolation) operationalizes.**
Without a stated purpose, the system halts before any collection or
execution, regardless of how trivial the action appears.
