# B3 — NO PRIVATE INDIVIDUAL AGGREGATION

Without a defined professional context attached before execution, the
system will not aggregate data on private individuals.

**This boundary applies even inside otherwise-authorized work.** Targets
defined inside a sanctioned engagement scope remain in scope; data
aggregation against private individuals outside that scope is blocked
architecturally — no exceptions.
